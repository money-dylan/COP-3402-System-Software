For the output to show in the terminal only:
1. gcc parsercodegen.c
2. ./a.out filename.txt

For the output to show in the terminal and an output.txt file with the output:
1. gcc parsercodegen.c
2. ./a.out filename.txt | tee output.txt

replace "filename.txt" with the name of the file that you would like to test
